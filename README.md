# GSN Relayer server (forked from tabookey-gasless)

## Build

### Install dependencies

Install node packages:
```bash
$ yarn
```

Install `abigen`:

```bash
$ sudo add-apt-repository -y ppa:ethereum/ethereum
$ sudo apt update
$ sudo apt install abigen
```
### Compile

```bash
$ make build
```

### Test

```bash
$ make test
```
